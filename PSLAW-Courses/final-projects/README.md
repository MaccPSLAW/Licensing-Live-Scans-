## Live Scan Integration

Live Scan fingerprinting stub is active in all HTML test files.
See `../live-scan/README.md` for full activation checklist.

**Activation when equipment arrives:**
1. Drop BSIS cert template → `PSLAW-Courses/live-scan/bsis_livescan_cert_template.pdf`
2. Update `LIVE_SCAN_CONFIG` in `../scripts/build_interactive_test_v2.js`
3. Set `enabled: true`, `showUploadSlot: true`, enter DOJ ORI and ATI prefix
4. Rebuild any module's HTML test to activate the upload slot
